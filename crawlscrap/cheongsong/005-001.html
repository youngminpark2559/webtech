005-001. selenium library @ # You should install selenium pip3 install selenium # To use selenium chrome browser, you should install chrome webdriver @ # BeautifulSoup proceeds request in backend # Selenium proceeds request via chrome browser # It means, if you don't hide browser window, it shows up to user # I access to web page via chrome browser # I bring all tags via css syntax or xpath syntax # I define, if tag has id=py, I order selenium to click that tag # Using BeautifulSoup and selenium is much beneficial @ # Let's try automatical clicks from selenium import webdriver import time browser = webdriver.Chrome("/Users/username/chromedriver") browser.get("http://python.org") # I use css selector # I get menus menus = browser.find_elements_by_css_selector('#top ul.menu li') pypi = None for m in menus: if m.text == "PyPI": pypi = m print(m.text) pypi.click() time.sleep(5) browser.quit()
