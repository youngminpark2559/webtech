007-001. understanding scraping web and crawling web @ web scraping : technology which makes computer to collect specific data on web by using specifying tags or regular expression web crawling(web spidering) : action that web crawler bot browses multiple web pages under defined rules In the end, entire step what we want is that 1. web crawler bot stores visited pages 1. we apply web scraping on stored pages to extract data what we want 1. we convert extracted data into dataset
