007-001. scraping web, crawling web @ Web scraping : technology which lets computer to collect data on web by using specifying tags or regular expression(offline, extract data from stored web page) Web crawling(web spidering) : action that web crawler bot browses multiple web pages under defined rules(online, bring web page) In the end, entire step of collecting data on web is that 1. web crawler bot stores visited pages 1. we apply 'web scraping' on stored pages to extract data what we want 1. we convert extracted data into dataset
